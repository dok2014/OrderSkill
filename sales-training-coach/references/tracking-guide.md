# 学习追踪指南

## 多维表格

- **应用**: 我的Claw销售课
- **链接**: https://my.feishu.cn/base/B1QjbuuXeaa8Lus9WtNcRNFanDh

## 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 学习日期 | 日期 | 课程推送日期 |
| 群聊名称 | 文本 | 推送群名称 |
| 执行时间 | 文本 | YYYY-MM-DD HH:mm 格式 |
| 课程类型 | 单选 | 晨课/午课 |
| 主题 | 文本 | 当天课程主题名 |
| 完成状态 | 单选 | 已推送/已完成/学员已回复/待评估 |
| 学习笔记 | 文本 | 学员回复或笔记内容（可选） |

## 自动记录流程

每次课程推送后自动执行：
1. 发送课程内容到群
2. 获取当前时间戳
3. 计算循环天次和主题
4. 插入一条新记录到表格

## 月度回顾

每月末执行：
1. 拉取当月所有记录
2. 统计完成率（已推送 vs 学员已回复）
3. 提取学习笔记中的模式
4. 生成月度进步报告
