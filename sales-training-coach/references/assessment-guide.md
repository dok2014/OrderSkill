# 评估反馈指南

## 评估原则

所有评估反馈统一以 **OrderFollower（初步跟单）** 方法论体系为标准基准。

> **OrderCoach 的评估本质**：检查学员是否能在实战中正确运用 OrderFollower 提供的方法论框架。

## 评估维度

晨课和午课使用统一的四维度评估框架，各维度参照的 OrderFollower 文件如下：

### 1. 框架符合度（0-10）
学员的回答是否遵循了对应方法论框架：
- 晨课 → 对应 OrderFollower 文件（见 `course-morning.md` 映射表）
- 午课 → 对应 OrderFollower 文件（见 `course-afternoon.md` 映射表）
- 推演复盘 → 参照 `02-sales-assistant.md` 的SOP流程

评分标准：
- 不使用框架 → 0-3
- 部分使用 → 4-6
- 基本遵循 → 7-8
- 完整运用 → 9-10

### 2. 专业度评分（0-10）
回答质量和可执行性：
- 逻辑清晰度：结构是否严谨
- 量化程度：是否用数据支撑
- 可行性：方案能否落地
- 完整性：是否覆盖所有关键点

### 3. 改进建议
- 指出1-2个最关键改进点
- 具体可操作，不说空话
- 如果回答已经很好，也要指出细节优化空间

### 4. 参考答案
- 给出经过优化的标准话术或方案
- 说明"为什么这个版本更好"
- 不要求学员照抄，但展示专业标杆

## 反馈格式

```
=== 四维度评估 ===

框架符合度：X/10
→ 评语

专业度评分：X/10
→ 评语

改进建议：
1. [具体建议]
2. [具体建议]

参考答案：
[优化后的版本]
```

## 进阶评估（每月/四周一次）

每月/每4个循环进行一次综合评估：
- 纵向对比：和一个月前的自己对比，哪些方面进步了？
- 能力雷达图：场景判断力 / 话术能力 / 工作习惯 / 认知水平 / 专业承诺 五个维度
- 下周/下月目标：基于当前水平设定可量化的下一个目标
