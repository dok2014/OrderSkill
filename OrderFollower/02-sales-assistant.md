# 🤖 AI销售跟单助手

## 🎯 核心理念：从"约束行为"到"提供洞察"

在AI时代，销售辅助工具的核心价值不再是强制销售人员"必须做什么"，而是通过智能分析提供"做什么可以提高赢单率"的洞察，甚至直接帮销售人员写好TODO-List，提示关键核查指标。

### 🔄 传统工具 vs AI驱动工具
| 传统销售工具 | AI销售跟单助手 |
|-------------|---------------|
| ❌ 强制填写表单 | ✅ 自动从对话中提取信息 |
| ❌ 固定流程约束 | ✅ 动态调整建议策略 |
| ❌ 被动记录数据 | ✅ 主动提供洞察建议 |
| ❌ 标准化报告 | ✅ 个性化行动指南 |
| ❌ 事后分析 | ✅ 实时预警和建议 |

    if timeline_too_aggressive:
        alerts.append("⏰ 项目时间表过于激进，建议重新评估可行性")

## 🛠️ 使用指南

### 启动条件
当以下任一条件满足时，启动AI销售跟单助手：
1. 销售机会通过资格认证，进入跟单阶段
2. 需要分析客户会议记录，提取关键信息
3. 面临竞争压力，需要制定应对策略
4. 决策进展缓慢，需要识别障碍
5. 准备重要客户沟通，需要个性化建议

### 三工具协同操作流程

#### 1. 飞龙虾（AI销售助手核心）
- **触发时机**：各销售阶段遇到障碍、需要策略建议时
- **核心功能**：
  - 会议记录智能分析（客户情绪、决策信号、未解决问题）
  - 决策链健康度实时评估
  - 竞争态势动态分析
  - 风险预警与机会窗口识别
  - 自动生成个性化TODO-List
- **输出**：结构化洞察建议，直接导入Feishu看板

#### 2. Trae（深度分析与报告）
- **触发时机**：需要复杂数据分析、模型构建、深度报告时
- **核心功能**：
  - 赢单率预测模型构建
  - 竞争对比分析报告
  - 客户行为模式分析
  - 销售漏斗优化模拟
- **输出**：深度分析报告、预测模型、优化建议

#### 3. Feishu（协作与可视化）
- **触发时机**：团队协作、进度跟踪、信息沉淀
- **核心功能**：
  - 销售看板（多维表格，实时更新各机会状态）
  - 决策链地图可视化
  - 风险预警仪表盘
  - 团队协作空间（文档、讨论、文件共享）
- **输出**：团队实时可见的销售仪表盘
