# 🦞 初步跟单：大客户销售全流程实战技能

## 🎯 技能定位

一套覆盖大客户销售**机会评估 → 跟单执行 → 售前协同 → 退出判断 → 专业承诺**全流程的实战技能体系。

适用于 ToB 大客户经理从接到线索到签约（或退出）的完整跟单生命周期。

## 🔄 跟单全流程链路

```
线索进来 → 机会评估（值不值得跟？）
                ↓
          评估通过 → 跟单执行（怎么推？）
                         ↓
                   需要售前 → 售前协同（怎么配合？）
                                  ↓
                   遇到瓶颈 → 专业承诺（拒绝还是谈判？）
                                  ↓
                   判断不匹配 → 退出决策（该不该撤？）
                                  ↓
                      ↓              ↓
                   继续推进        礼貌退出，保持关系
```

## 🛠️ 使用指南

### 策略输出格式

每条建议遵循 **SPIN + FABE** 框架：
1. **归纳核心问题** → 客户当前卡在哪？
2. **假设可能场景** → 分析几种可能的深层原因
3. **举例落地方法** → 给可以直接用的话术和动作
4. **效果检查标准** → 可量化、可验证的成功标准

### 执行追踪原则

- 核心事项重复叮嘱，不厌其烦
- 要求及时反馈，确保执行力不打折
- 效果检查标准要落实到位，有追踪才有闭环

### 沟通风格

- ToB 销售场景：严谨专业，不编造数据，不夸大效果
- 沟通咨询场景：幽默活泼，非暴力沟通，先共情再结构
- 降维沟通原则：永远从「我具体帮你做什么」出发

## 📋 文件索引

### 01-机会评估框架
**文件：** `01-opportunity-evaluation.md`
**定位：** 判断机会价值与可行性，决定是否投入资源
**核心方法：** BANT+MEDDIC 资格认证、决策链分析、竞争分析、风险评估
**关键产出：** Go（进入跟单）/ Watch（观察等待）/ No-Go（退出）

### 02-跟单助手
**文件：** `02-sales-assistant.md`
**定位：** 推进跟单执行，提供洞察、TODO-List、风险预警
**核心方法：** 决策链健康度分析、竞争态势洞察、自动生成行动清单
**关键产出：** 结构化洞察报告、TODO-List、风险预警

### 03-售前协同策略
**文件：** `03-sales-presales-collaboration.md`
**定位：** 销售×售前配合的原则、流程、话术
**核心规则：** 销售控场，售前输出，互相补位，绝不拆台
**关键产出：** 信息交底清单、会前检查清单、会后闭环模板

### 04-退出决策分析
**文件：** `04-exit-decision.md`
**定位：** 预算与技术要求不匹配时的退出判断
**核心判断：** 投入产出比 < 可接受阈值 → 退出
**关键产出：** 退出信号识别、退出话术、关系维护策略

### 05-专业承诺与委婉拒绝
**文件：** `05-professional-commitment.md`
**定位：** 从"老好人"到"可预测的专业人士"

**关键产出：** 场景化话术、条件谈判策略、底线原则
